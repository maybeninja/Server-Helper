# Server Helper


## 🚧 Upcoming Features

### 🧹 Advanced Purge System
- Delete messages using Message ID  
- Filtered purge options:
  - Bot messages  
  - User messages  
  - Embedded messages  
  - Links / URLs  
  - Attachments  

---

### 📝 Change Log System
- Dedicated logging category  
- Separate channels for:
  - Message edits & deletions  
  - Role updates  
  - Channel updates  
  - Member activity  

---

### 🔍 OCR + Text Scanner
- Extract text from images  
- Detect hidden or bypassed content  

---

### 🚫 Profanity Filter with Auto-Warn
- Custom banned words system  
- Detect in text and OCR content  
- Automatic warnings  

---

### ⚖️ Punishment Pipeline
- Configurable escalation system:
  - Warn → Mute → Kick → Ban  
- Custom thresholds  
- Optional warning decay  

---

### 📩 Appeal System
- Users can appeal moderation actions  
- Appeals sent to a review channel  

---

### 🖼️ Media Moderation
- Image scanning for restricted content  
- Sticker analysis (where supported)
